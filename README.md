> **Your central hub for Resident Evil 4 modding. Browse, download, and manage 70+ mods in one place.**

## ✨ Features

- 🔧 **70+ Mods** - Gameplay mods, texture packs, models
- 🎯 **Smart Filters** - By version (UHD, 2007, PS3) and category
- ⚡ **One-Click Downloads** - Direct MediaFire & Mega links
- 📥 **Download Tracking** - Per-user and per-mod statistics
- 🔐 **Auto-Login** - Encrypted credential storage
- 🎨 **Modern Dark UI** - Pink/Purple theme with animations
- 🔊 **Sound Effects** - Professional encrypted sound pack
- 🔄 **Auto-Updater** - Cloud-based updates via Supabase
- 👥 **Role System** - Owner, Admin, Moderator, Member
- 🛡️ **Moderator Panel** - User management tools

## 📥 Installation

1. Download `RE_HUB.rar` from [Releases]([https://github.com/TEX-Rikka/RE-HUB/releases](https://github.com/Tex001222/RE-HUB-V1.0.0/releases/tag/FirstVersion))
2. Extract the ZIP file
3. Run `RE_HUB.exe`
4. Create an account or login
5. Start browsing mods!
